


# Time Tree v1



![Time Tree!](time-tree-demo.png)



## About
- Peter's Conception of a Time Tree



## Running
- Check *Releases* (if available)
- Pull Project
- Download *Godot* **`3.2`**
- Open Project in Godot



## Controls
- **`ESC`:** Open Pause Menu
- **`W/↑`:** Move Up
- **`S/↓`:** Move Down
- **`A/←`:** Move Left
- **`D/→`:** Move Right
- **`_/PgUp`:** (Battle) Return to Chrono-Tree
- **`Enter/PgDn`:** (Chrono-Tree) Enter selected Battle Node
